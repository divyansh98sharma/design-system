    </div>

    <div className="gs-step-num">2</div>
    <div className="gs-step-body">
      <h3>Import design tokens</h3>
      <p>
        Add the token stylesheet to your application's global styles. The tokens file
        includes primitives, semantic aliases, and component-scoped variables — all as
        CSS custom properties on <code>:root</code>.
      </p>

```scss
/* styles.scss */
@use 'path/to/design-system/src/ds/tokens/tokens';

/* or add to angular.json styles array */
// "src/ds/tokens/tokens.scss"
```

      <p>Once imported, every CSS custom property is available globally:</p>

```scss
.my-element {
  color: var(--color-user-base);
  font-size: var(--font-size-14);
  border-radius: var(--radius-md);
}
```

    </div>
  </div>

  <div className="gs-step">
    <div className="gs-step-num">3</div>
    <div className="gs-step-body">
      <h3>Import a component</h3>
      <p>
        All components are standalone Angular components. Import them directly into any
        <code>@Component</code> or <code>@NgModule</code> <code>imports</code> array.
      </p>

```typescript
// my-feature.component.ts
import { ButtonComponent } from '@ds/button';

@Component({
  selector: 'app-my-feature',
  standalone: true,
  imports: [ButtonComponent],
  template: `
    <ds-button label="Save" type="primary"></ds-button>
  `,
})
export class MyFeatureComponent {}
```

    </div>
  </div>

## Storybook tips

<div className="gs-callout">
  <strong>Controls panel:</strong> Select any story, open the <em>Controls</em> tab at the
  bottom of the canvas, and tweak props live. Changes are reflected in the URL so you can
  share a specific state with a team-mate.
</div>

<div className="gs-callout">
  <strong>A11y panel:</strong> The <em>Accessibility</em> tab runs axe-core on every render
  and surfaces WCAG violations automatically. Fix the red items before shipping.
</div>

<div className="gs-callout">
  <strong>Chromatic:</strong> Every PR runs visual regression tests against the last approved
  baseline. Accept or reject snapshot changes in the Chromatic UI before merging.
</div>
